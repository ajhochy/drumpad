# Repo map

- `index.html` — single-page shell, tabs, drumrot reveal overlay
- `css/main.css` — chassis + non-card styles
- `css/cards.css` — v0.3 card chrome (drumrot-card, tier-*, portrait, etc.)
- `js/`
  - `main.js` — bootstrap
  - `drumrots.js` — TIER_CONFIG, DRUMROTS roster, rollDrumrot, renderDrumrotCard, drops tab, reveal overlay
  - `achievements.js`, `library.js`, `builder.js`, `highway.js`, `lessons.js`, `state.js`, `audio.js`, `scoring.js`, `midi-*.js`
- `art/drumrots/` — voxel renders, one .webp per drumrot id
- `brainrots/` — parody source images
- `scripts/` — generators, install/start helpers
